Generation and Analysis of Directed Maze with Circuits
Procedural Content Generation for Games (Spring 2015)
A wall carving algorithm which is adapted from depth-first search algorithm is proposed to generate directed mazes with single solution path, no dead ends, no isolated nodes, and several circuits. In addition, an analysis method is utilized to evaluate if the generated mazes help player explore most of themselves without losing patience.
Curve and Surface Modeling
Geometric Modeling (Spring 2014)
User can interactively generate various curves or surface, such as Bezier, cubic B-spline, and subdivision curves/surfaces. User can also reconstruct curve based on NN-Crust or Crust algorithm.
Particle-Based Fluid Simulation
Computer Game and Animation Techniques (Spring 2014)
Smoothed particle hydrodynamics (SPH) is a method to simulate fluid based on particle systems. It supposes that the physical quantities of each particle can be interpolated by centain radial symmetrical smoothing kernels. For this project, SPH is applied to simulate the movement of water being poured into a container.

Kinect Motion Capture
Physically Based Modeling for Interactive Simulation and Games (Spring 2011)
The system can be utilized to capture character motion by Microsoft Kinect. User only needs to perform a calibration pose in the first place, and then the system starts to capture the skeleton and motion information. After the capture process, the system can produce motion data for advanced use.
Kinect Inverse Kinematics
Computer Animation and Simulation (Spring 2011)
This system utilizes Cyclic Coordinate Descent method to solve inverse kinematics problem. Via a natural interface based on Microsoft Kinect, user can manipulate this system without any mouse or keyboard commands. Some parameters can be adjusted by stepping away from the origin, e.g., number of joints, bone length, and joint angle.
Processing Particle System
Computer Animation and Simulation (Spring 2011)

Constrained Particle System
Physically Based Modeling for Interactive Simulation and Games (Spring 2011)
This is a physically-based simulation of a 1D deformable chain-like structure modeled as a particle system. The chain is "hard" constrained, e.g., rigid edge constraint, pin constraint, and ring constraint. User can interactively adjust parameters and apply extra force by arrow keys and mouse, respectively.
Jello Cube Simulation
Physically Based Modeling for Interactive Simulation and Games (Spring 2011)
This is a physically-based simulation of a jello cube modeled as a three-dimensional mass-spring system. The cube is constrained to stay inside a bounding box with inclined plane, sphere, cone, and a time-independent force field. User can interactively adjust parameters and apply extra force by arrow keys and mouse, respectively.
